Preparacion Datos

4_ProcesarDatos.R

Procesamiento de Datos

Este script se encarga de cargar, transformar, fusionar y enriquecer los datos

Carga de Datos

Función Inicial: El script inicia cargando los datos básicos de producción desde archivos CSV:

Olanet_Producido <- CargarCSV("./OlanetDatos/Olanet_Producido.csv")
Olanet_Planificado <- CargarCSV("./OlanetDatos/Olanet_Planificado.csv")

Limpieza y Transformación Inicial

Normalización de identificadores: Se normalizan los campos EXPEDIENTE y OF para asegurar que sean numéricos:

Olanet_Producido$EXPEDIENTE <- as.numeric(gsub("[^0-9]", "", Olanet_Producido$EXPEDIENTE))
Olanet_Planificado$EXPEDIENTE <- as.numeric(gsub("[^0-9]", "", Olanet_Planificado$EXPEDIENTE))

Olanet_Producido$OF <- as.numeric(gsub("[^0-9]", "", Olanet_Producido$OF))
Olanet_Planificado$OF <- as.numeric(gsub("[^0-9]", "", Olanet_Planificado$OF))

Esta normalización elimina cualquier carácter no numérico y convierte los valores a tipo numérico, facilitando las operaciones posteriores.

Agregación de Datos Producidos

Creación de resumen por claves: Se agregan los datos de producción por claves principales:

Olanet_Producido_Agg <- Olanet_Producido %>%
  mutate(
    FechaInicio_pr = as.POSIXct(FechaInicio_pr, format = "%Y-%m-%d %H:%M:%OS"),
    FechaFin_pr = as.POSIXct(FechaFin_pr, format = "%Y-%m-%d %H:%M:%OS")
  ) %>%
  group_by(EXPEDIENTE, REFERENCIA, OF, FASE, MAQUINA) %>%
  summarise(...)

Este proceso:

  • Convierte fechas al formato POSIXct para manipulación temporal

  • Agrupa por las claves principales de negocio

  • Calcula métricas agregadas como fechas mínimas/máximas y sumas de cantidades

Calculo de Tiempos Operativos

Tiempos de Producción Se calculan los tiempos efectivos de producción:

DF_En_Produccion <- Olanet_Producido %>%
  filter(Estado == "PRODUCCION") %>%  
  group_by(OF, FASE, MAQUINA, REFERENCIA, EXPEDIENTE) %>%
  summarise(
    Tiempo_Total_Produccion_min = sum(Duracion_min, na.rm = TRUE), 
    .groups = "drop"
  )

Tiempos de Preparación Se calculan los tiempos de preparación:

DF_En_Preparacion <- Olanet_Producido %>%
  filter(Estado == "PREPARACION") %>%  
  group_by(OF, FASE, MAQUINA, REFERENCIA, EXPEDIENTE) %>%
  summarise(
    TiempoRealPreparacion_min = sum(Duracion_min, na.rm = TRUE), 
    .groups = "drop"
  )

Estos cálculos separan los tiempos por tipo de actividad para análisis detallados.

Fusión y Cálculo de KPIs

Creación del Dataset Completo Se fusionan los datos planificados y reales, calculando métricas de rendimiento:

pr_COMPLETO <- Olanet_Planificado %>%
  mutate(
    FechaInicio_Plani = as.POSIXct(paste(FechaInicio_Plani, "00:00:00"), format = "%Y-%m-%d %H:%M:%S"),
    FechaFin_Plani = as.POSIXct(paste(FechaFin_Plani, "00:00:00"), format = "%Y-%m-%d %H:%M:%S")
  ) %>%
  left_join(Olanet_Producido_Agg, by = c("EXPEDIENTE", "REFERENCIA", "OF", "FASE", "MAQUINA")) %>%
  left_join(DF_En_Produccion, by = c("EXPEDIENTE", "REFERENCIA", "OF", "FASE", "MAQUINA")) %>%
  left_join(DF_En_Preparacion, by = c("EXPEDIENTE", "REFERENCIA", "OF", "FASE", "MAQUINA")) %>% 
  mutate(
    FechaInicio_pr_trunc = as.Date(FechaInicio_pr),
    FechaFin_pr_trunc = as.Date(FechaFin_pr),
    FechaInicio_Plani_trunc = as.Date(FechaInicio_Plani),
    FechaFin_Plani_trunc = as.Date(FechaFin_Plani),
    
    DiasDiferenciaInicio = as.numeric(FechaInicio_pr_trunc - FechaInicio_Plani_trunc),
    DiasDiferenciaFin = as.numeric(FechaFin_pr_trunc - FechaFin_Plani_trunc),
    
    CantidadPendiente = CantidadPrevista - CantidadRealizada,
    CompletadoCantidad = ifelse(CantidadPrevista > 0, coalesce(round(CantidadRealizada / CantidadPrevista, 2), 0), 0),
    
    TiempoProduccionReal_min = TiempoReal_min,
    TiempoProduccionPlanificado_min = (TiempoPreparacion*CantidadPrevista) / 60 + (TiempoTeorico*60),
    
    TiempoPrevistoPreparacionUnitario_min = TiempoPreparacion,
    
    TiempoRealPreparacion_min = coalesce(TiempoRealPreparacion_min, 0), 
    TiempoPrevistoPreparacion_min = (TiempoPreparacion * CantidadPrevista) / 60 ,

    TiempoPrevistoProduccion_min = TiempoTeorico*60,
    TiempoProduccion_min = Tiempo_Total_Produccion_min,
    
    TiempoPrevistoUnitario_min = TiempoCiclo, 
    TiempoProducidoUnitario_min = ifelse(CantidadRealizada > 0, Tiempo_Total_Produccion_min / CantidadRealizada, NA),

    Disponibilidad = ifelse(Tiempo_Total_Produccion_min > 0, format(round((TiempoReal_min / Tiempo_Total_Produccion_min) * 100, 2), decimal.mark = ","), NA),  
    Calidad = ifelse((CantidadRealizada + CantidadMalas) > 0, (CantidadRealizada / (CantidadRealizada + CantidadMalas)) * 100, NA),
    Productividad =ifelse(CantidadPrevista > 0, format(round((CantidadRealizada / CantidadPrevista) * 100, 2), decimal.mark = ","), NA)         
  ) %>%
  select(
    EXPEDIENTE, REFERENCIA, OF, FASE, FASE_DESCR, MATERIAL_ENTRADA, MATERIAL_ENTRADA_DESCR,MATERIAL_SALIDA, MAQUINA, MAQ_DESC, ESTADO_OLANET, 
    FechaInicio_Plani, FechaInicio_pr, DiasDiferenciaInicio, FechaFin_Plani, FechaFin_pr, DiasDiferenciaFin, 
    CantidadPrevista, CantidadRealizada, CantidadPendiente, CompletadoCantidad, CantidadMalas, 
    TiempoProduccionPlanificado_min,TiempoProduccionReal_min, 
    TiempoPrevistoPreparacionUnitario_min, 
    TiempoPrevistoPreparacion_min, TiempoRealPreparacion_min, 
    TiempoPrevistoProduccion_min, TiempoProduccion_min, 
    TiempoPrevistoUnitario_min, TiempoProducidoUnitario_min
  )
pr_COMPLETO <- pr_COMPLETO[, !(names(pr_COMPLETO) %in% c("MATERIAL_ENTRADA", "MATERIAL_ENTRADA_DESCR"))]
pr_COMPLETO <- unique(pr_COMPLETO)

Cálculos de KPIs importantes:

  • Diferencias en días entre fechas planificadas y reales

  • Cantidades pendientes y porcentajes de completitud

  • Tiempos de producción real vs. planificado

  • Indicadores de rendimiento: Disponibilidad, Calidad y Productividad

Enriquecimiento para Producción

Enriquecimiento del Dataset Original Se añaden datos de planificación al dataset original:

Olanet_Planificado_Key <- pr_COMPLETO %>%
  select(OF, FASE, MAQUINA, REFERENCIA, EXPEDIENTE, CantidadPrevista, 
         TiempoProduccionPlanificado_min, TiempoPrevistoPreparacion_min, 
         TiempoPrevistoProduccion_min) %>%
  distinct()

Olanet_Producido <- Olanet_Producido %>%
  left_join(Olanet_Planificado_Key, by = c("OF", "FASE", "MAQUINA", "REFERENCIA", "EXPEDIENTE"))

Guardar Datos creados

Exportación de Resultados Los datasets procesados se guardan en archivos CSV:

GuardarCSV(Olanet_Producido, "Olanet_Producido", "./OlanetDatos")
GuardarCSV(pr_COMPLETO, "Olanet_Completo", "./OlanetDatos")

5_CrearMaestros.R

Creación de Maestros Identificadores para Modelo en Estrella

Este script genera tablas maestras para la gestión de identificadores y categorías.

Carga de Datos Base

Carga del Dataset Planificado Se carga el dataset de planificación para extraer datos maestros:

Olanet_Planificado <- CargarCSV("./OlanetDatos/Olanet_Planificado.csv")

Generación de Tablas Maestras

Extracción de Identificadores Únicos Se crean tablas maestras para las principales entidades con registros unicos:

Maestro_Maquina <- Olanet_Planificado %>%
  select(MAQUINA, MAQ_DESC) %>%
  drop_na()  %>% 
  distinct()
Maestro_Expediente <- Olanet_Planificado %>%
  select(EXPEDIENTE) %>%
  drop_na()  %>% 
  distinct()
Maestro_OF <- Olanet_Planificado %>%
  select(OF) %>%
  drop_na() %>%
  distinct() 
Maestro_Referencia <- Olanet_Planificado %>%
  select(REFERENCIA) %>%
  drop_na() %>%
  distinct() 
Maestro_EstadoOlanet <- Olanet_Planificado %>%
  select(ESTADO_OLANET) %>%
  drop_na()  %>% 
  distinct()

Exportación de Maestros

Guardado de Tablas Maestras Se utiliza un bucle para guardar todas las tablas maestras:

XX_MAESTROS_OLANET <- list(
  Maestro_Maquina = "Maestro_Maquina",
  Maestro_Expediente = "Maestro_Expediente",
  Maestro_OF = "Maestro_OF",
  Maestro_Referencia = "Maestro_Referencia",
  Maestro_EstadoOlanet = "Maestro_EstadoOlanet"
)

for (nombre in names(XX_MAESTROS_OLANET)) {
  GuardarCSV(get(nombre), XX_MAESTROS_OLANET[[nombre]], "./OlanetDatos/MaestrosIdentificadores")
}
