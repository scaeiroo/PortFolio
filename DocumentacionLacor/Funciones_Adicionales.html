Función de Configuración

1.1 CargarConfiguracion
La función CargarConfiguracion() carga y filtra las configuraciones de una base de datos en formato JSON para ajustarlas a las preferencias de base de datos, alias y dominio. La configuración se almacena globalmente para ser utilizada en otros scripts.

CargarConfiguracion <- function(FicheroConf = './Config/Config.json', TipoDBElegido ='SQL Server', AliasElegido = 'DBA', DominioElegido = 'Remoto'){
  .GlobalEnv$configuracion <- fromJSON(FicheroConf)
  .GlobalEnv$configuracion <- .GlobalEnv$configuracion %>% filter((toupper(TipoDB) == toupper(TipoDBElegido))) 
  .GlobalEnv$configuracion <- .GlobalEnv$configuracion %>% filter((toupper(Alias) == toupper(AliasElegido))) 
  .GlobalEnv$configuracion <- .GlobalEnv$configuracion %>% filter((toupper(Dominio) == toupper(DominioElegido)))
  
  return(.GlobalEnv$configuracion)
}

Parámetros:

  • FicheroConf: Ruta al archivo de configuración (por defecto ‘./Config/Config.json’).

  • TipoDBElegido, AliasElegido, DominioElegido: Parámetros para filtrar las configuraciones.

Función de Guardado

1.4 GuardarCSV
La función GuardarCSV() Guarda un DataFrame como archivo CSV y verifica que se guardó correctamente.

GuardarCSV <- function(df, nombre_archivo, ruta) {
  tryCatch({
    if (!dir.exists(ruta)) {
      dir.create(ruta, recursive = TRUE) 
    }
    ruta_completa <- file.path(ruta, paste0(nombre_archivo, ".csv"))
    write.csv(df, ruta_completa, row.names = FALSE, fileEncoding = "UTF-8", na = "")
    cat("✅ Guardado correctamente:", ruta_completa, "- Registros:", nrow(df), "\n")
  }, error = function(e) {
    cat("❌ Error al guardar:", nombre_archivo, "- Mensaje:", e$message, "\n")
  })
}

Parámetros:

  • df: El DataFrame a guardar.

  • nombre_archivo: El nombre del archivo.

  • ruta: La ruta donde guardar el archivo.

Función de Guardado RData

1.5 GuardarRData
La función GuardarRData() Guarda un objeto R en un archivo .RData y verifica que se haya guardado correctamente.

GuardarRData <- function(objeto, ruta_archivo) {
  tryCatch({
    nombre_objeto <- deparse(substitute(objeto))
    assign(nombre_objeto, objeto)
    save(list = nombre_objeto, file = ruta_archivo)
    if (file.exists(ruta_archivo)) {
      registros <- if (is.data.frame(objeto)) nrow(objeto) else "N/A"
      cat("✅ Archivo guardado: '", ruta_archivo, "' - Objeto: ", nombre_objeto, " - Registros: ", registros, "\n")
    } else {
      cat("❌ Error: No se pudo guardar '", ruta_archivo, "'.\n")
    }
  }, error = function(e) {
    cat("❌ Error al guardar '", ruta_archivo, "': ", e$message, "\n")
  })
}

Parámetros:

  • objeto: El objeto R que deseas guardar (como un DataFrame, lista, etc.).

  • ruta_archivo: La ruta completa y nombre del archivo .RData donde se guardará el objeto.

Función de Carga de CSV

1.6 CargarCSV
La función CargarCSV() Carga un archivo CSV y maneja los casos donde el archivo no existe o el formato no es válido

CargarCSV <- function(ruta_archivo) {
  if (!file.exists(ruta_archivo)) {
    cat("❌ El archivo CSV '", ruta_archivo, "' no se encuentra. Verifica el nombre o la fecha.\n")
    stop("Archivo CSV no encontrado.")
  }
  
  tryCatch({
    df <- read.csv(ruta_archivo, sep = ",", stringsAsFactors = FALSE, fileEncoding = "UTF-8")
    if (ncol(df) == 1) {
      df <- read.csv(ruta_archivo, sep = ";", stringsAsFactors = FALSE, fileEncoding = "UTF-8")
    }
    cat("✅ El archivo CSV '", ruta_archivo, "' se cargó correctamente. Registros: ", nrow(df), " - Columnas: ", ncol(df), "\n")
    return(df)
  }, error = function(e) {
    cat("❌ Error al cargar '", ruta_archivo, "': ", e$message, "\n")
    stop(e)
  })
}

Parámetros:

  • ruta_archivo: La ruta completa del archivo CSV que deseas cargar.

La función CargarRData() Carga un archivo .RData y verifica que el proceso se haya realizado correctamente.

CargarRData <- function(ruta_archivo) {
  if (!file.exists(ruta_archivo)) {
    cat("❌ El archivo RData '", ruta_archivo, "' no se encuentra. Verifica el nombre o la fecha.\n")
    stop("Archivo RData no encontrado.")
  }
  objetos_cargados <- load(ruta_archivo, envir = .GlobalEnv)  
  cat("✅ El archivo RData '", ruta_archivo, "' se cargó correctamente. Contiene: ", paste(objetos_cargados, collapse = ", "), "\n")
  if (length(objetos_cargados) == 1) {
    return(get(objetos_cargados[1], envir = .GlobalEnv))
  } else {
    return(mget(objetos_cargados, envir = .GlobalEnv))
  }
}

Parámetros:

  • ruta_archivo: La ruta completa del archivo .RData que deseas cargar.

Función Obtener Última Fecha de Actualización

2.2 ObtenerUltimaFechaAct
Esta función ObtenerUltimaFechaAct() revisa los archivos de producción diaria almacenados en la carpeta de salida (./Output/) y devuelve la fecha más reciente de los archivos guardados. Es útil para conocer la última actualización de datos disponible en el sistema.

ObtenerUltimaFechaAct <- function() {
  archivo_total <- './Output/Produccion_Diario_TOTAL.RData'
  
  if (!file.exists(archivo_total)) {
    return(NULL) 
  }
  
  ProduccionDiario_TOTAL_DF <- get(load(archivo_total))
  
  if (nrow(ProduccionDiario_TOTAL_DF) > 0 && "Jornada" %in% colnames(ProduccionDiario_TOTAL_DF)) {
    return(max(ProduccionDiario_TOTAL_DF$Jornada, na.rm = TRUE))
  } else {
    warning("El archivo total está vacío o no contiene la columna 'Jornada'.")
    return(NULL)
  }
}

Parámetros: - No requiere parámetros de entrada.

Función Filtrar Duplicados

2.3 FiltrarDuplicadosTOTAL
La función FiltrarDuplicadosTOTAL() elimina registros duplicados o no válidos del conjunto de datos de producción, priorizando registros con incidencias, fechas de fin más recientes y mayores cantidades buenas.

FiltrarDuplicadosTOTAL <- function(df) {
  if (!dir.exists("./Output")) {
    dir.create("./Output")
  }
  df <- df %>%
    filter(!(Estado == "PARO" & is.na(Operario) & is.na(EXPEDIENTE) & is.na(REFERENCIA) & is.na(OF)))
  
  df <- df %>%
    group_by(OF, FASE, EXPEDIENTE, REFERENCIA, MAQUINA, FechaInicio_pr, FechaFin_pr) %>%
    filter(!(is.na(MATERIAL_SALIDA) | MATERIAL_SALIDA == "")) %>%
    ungroup()
  
  df_filtrado <- df %>%
    group_by(OF, FechaInicio_pr, Estado, FASE, MATERIAL_SALIDA) %>% 
    mutate(IncidenciaPrioridad = !is.na(Incidencia),
           OperarioSeleccionado = coalesce(first(Operario), NA)) %>%  
    slice_max(order_by = IncidenciaPrioridad, with_ties = TRUE) %>% 
    slice_max(order_by = FechaFin_pr, with_ties = TRUE) %>%
    slice_max(order_by = Cantidad_Buenas, with_ties = FALSE) %>%
    select(-IncidenciaPrioridad, -OperarioSeleccionado) %>%
    ungroup()
  
  filas_eliminadas <- nrow(df) - nrow(df_filtrado)
  
  cat("🔄 Total de filas originales: ", nrow(df), "\n")
  cat("🎉 Total de filas después del filtrado: ", nrow(df_filtrado), "\n")
  cat("🔥 Filas eliminadas: ", filas_eliminadas, "\n")
  
  return(df_filtrado)
}

Parámetros:

  • df: El DataFrame del que deseas eliminar los duplicados.

Función Filtrar Duplicados TOTAL

2.3 FiltrarDuplicadosTOTALDIARIO
La función FiltrarDuplicadosTOTALDIARIO() optimiza el proceso de filtrado de duplicados al trabajar solo con un subconjunto de los datos totales, enfocándose en los registros recientes para mejorar el rendimiento.

FiltrarDuplicadosTOTALDIARIO <- function(df_total, df_diario) {
  n_filas_a_registrar <- nrow(df_diario) * 2
  filas_a_registrar <- tail(df_total, n_filas_a_registrar)
  filas_a_registrar <- filas_a_registrar %>%
    filter(!(Estado == "PARO" & is.na(Operario) & is.na(EXPEDIENTE) & is.na(REFERENCIA) & is.na(OF)))
  
  filas_a_registrar <- filas_a_registrar %>%
    group_by(OF, FASE, EXPEDIENTE, REFERENCIA, MAQUINA, FechaInicio_pr, FechaFin_pr) %>%
    filter(!(is.na(MATERIAL_SALIDA) | MATERIAL_SALIDA == "")) %>%
    ungroup()
  
  df_filtrado <- filas_a_registrar %>%
    group_by(OF, FechaInicio_pr, Estado, FASE, MATERIAL_SALIDA) %>% 
    mutate(IncidenciaPrioridad = !is.na(Incidencia),
           OperarioSeleccionado = coalesce(first(Operario), NA)) %>%  
    slice_max(order_by = IncidenciaPrioridad, with_ties = TRUE) %>% 
    slice_max(order_by = FechaFin_pr, with_ties = TRUE) %>%
    slice_max(order_by = Cantidad_Buenas, with_ties = FALSE) %>%
    select(-IncidenciaPrioridad, -OperarioSeleccionado) %>%
    ungroup()
  
  df_sin_filas_a_registrar <- df_total %>%
    anti_join(filas_a_registrar, by = c("OF", "FechaInicio_pr", "Estado", "FASE", "MATERIAL_SALIDA"))
  df_actualizado <- bind_rows(df_sin_filas_a_registrar, df_filtrado)
  filas_eliminadas <- nrow(df_total) - nrow(df_actualizado)
  
  cat("🔄 Total de filas originales: ", nrow(df_total), "\n")
  cat("🎉 Total de filas después del filtrado: ", nrow(df_actualizado), "\n")
  cat("🔥 Filas eliminadas o reemplazadas: ", filas_eliminadas, "\n")
  
  
  return(df_actualizado)
}

Parámetros:

  • df_total: El DataFrame total que contiene todos los registros históricos
  • df_diario: El DataFrame con los nuevos registros diarios

      "SELECT  
            SmFase.smFasePersonalizado1Txt AS 'EXPEDIENTE',
            smOrdenNme AS 'REFERENCIA', 
            SmMaterial.smMaterialId AS 'MATERIAL_SALIDA', 
            SmMaterial.smMaterialNme AS 'MATERIAL_SALIDA_DESCR', 
            smOrdenId AS 'OF',

            smFaseNme AS 'FASE_DESCR', 
            smNodo.smNodoId AS 'MAQUINA',  
            smNodo.smNodoNme AS 'MAQ_DESC',
            jornadaDte AS Jornada, 
            OLANET.ObtenerTurnoId(smCalendarioHorarioDetalle.smTurnoUid) AS Turno, 
            ISNULL(ISNULL(ISNULL(shNodoEstadoOperario.FechaInicioLocalDte, shNodoEstadoOperario.FechaInicioDte), ISNULL(shNodoEstadoFase.FechaInicioLocalDte, shNodoEstadoFase.FechaInicioDte)), ISNULL(shNodoEstado.FechaInicioLocalDte, shNodoEstado.FechaInicioDte)) AS FechaInicio_pr,
